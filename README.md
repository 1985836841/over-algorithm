## 说明

算法章节采用`Go`语言作为伪代码，因为Go简单灵活，不具备Go语言基础也能看懂代码。且语法极度简洁，且具备指针设计，极符合当代数据结构与算法学习要求。    

[source](https://github.com/overnote/five-x/tree/master/sources)文件夹下也提供了C/C++版本，未来会增加：Java，JavaScript，Rust版本。

#### 1.2 算法入门
