## 说明

算法章节采用`Go`语言作为伪代码示例，因为Go简单灵活，读者不具备Go语言基础也能看懂代码，同时提供完整的C语言版本。  

当然，[source](https://github.com/overnote/five-x/tree/master/sources)文件夹下拥有各类语言版本的实现案例，未来会不断涵盖Java，JavaScript，TypeScript，C，C++等各个版本。
